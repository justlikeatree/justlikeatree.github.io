github+hexo搭建博客记录

（vmware12虚拟机里的）win7x64位系统，github，nodejs，hexo

一、安装及环境配置

1、安装github,nodejs

github,nodejs在官网上都有安装的压缩文件下载，位置随意，我都安装在D盘上，直接按安装程序直接next下去。

(github,nodejs的安装可参考hexo官网文档，里面提供了安装包的链接和方法，我是下载安装程序来安装的)

安装好后，查看是否安装成功，显示安装版本。

  • 同时按一下键盘上的“win””R”键，出现运行窗口，输入“cmd”，再确定。
  • 下面的代码分三次输入，每次一行，每次输入后显示出对应版本即安装成功

git --version

node -v

npm -v

2、github账号注册申请

  • 点击标题后会进入github账号申请的页面,注册时记住用户名(username)、邮箱(email)、密码(password)
  • 新建一个仓库(new repository),在repository name下写上命名username.github.io（username是注册时的用户名 eg:justlikeatree.github.io）

然后，点击最左下方的绿色按钮create repository

  • 建好后点击ssh,再点击这行填写栏后面的图标，复制填写栏里的内容

    3、配置NodeJS环境

  • 参考这个网址

    4、安装hexo

  • 自己随便在哪个盘上新建一个名为hexo的空文件夹作为hexo的文件安装路径，我是在D盘上新建的
  • 同时按”win”、“R”，输入cmd
  • 在命令行中输入npm install hexo-cli -g,执行后再输入npm install hexo –save，等待安装完成
  • 安装好后，鼠标右键单击之前新建的hexo文件夹，选择Git Bash Here
  • 在出现的界面的$ 后面输入hexo init，初始化hexo（要出现$ 且可编辑状态的时候才算安装完成）
  • 在$ 后输入hexo -v，检测版本，看是否安装成功

    二、编写及发布到网上

    1、编辑_config.yml

  • 打开之前建的hexo文件夹，里面已经有了内容，找到并编辑_config.yml，
    如果用电脑自带的记事本的话，一定要注意另存为时，不仅是文件名，编码也要改为

    UTF-8编码！UTF-8编码！UTF-8编码！

    否则有些中文会显示成 乱码！最好用其他的编辑器打开。

编辑里面的内容时，

所有冒号后面一定要有一个空格！冒号后面一定要有一个空格！冒号后面一定要有一个空格！

  • 设置主题

    • 在hexo里找到想要的主题，点击看中的主题下面的字，我选的是yelee。进入github里面，点击右边绿色的下载，下载zip文件。
    • 将文件解压后，放在hexo文件夹下的themes文件里。
    • 回到hexo文件夹下的_config.yml，修改theme

      theme: yelee //刚刚安装的主题名称

    （注意看主题的文件夹里的language的语言名字是什么，在_config.yml里的language: 后面写上语言）

  • 修改博客

    title: 挣扎纪 //博客名字
    subtitle: 学习啊学习啊啊啊 //博客描述
    description: 经常性发呆，间接性发奋 //个人描述
    keywords:
    author: 郭啥啥 //作者
    language: default
    timezone:

  • 本地查看
    • 鼠标右键单击hexo文件夹，选择Git Bash Here，进入Git命令页面，输入hexo g，运行完后再输入hexo s。
    • 打开浏览器，输入http://localhost:4000/ ,可以看到博客的样子。要关掉Git Bash的窗口的话，要先按ctrl+c。

      2、发布到网上

  • 打开_config.yml文档，修改最后面的内容如下，没有的直接添上,保存。

    deploy:
    type: git //使用Git 发布
    repo: git@github.com:justlikeatree/justlikeatree.github.io.git // 刚创建的Github仓库
    branch: master

  • 鼠标右键单击hexo文件夹，选择Git Bash Here，进入Git命令页面，设置git身份信息，输入

git config –global user.name “你的用户名” //这两行都只修改双引号里的内容
git config –global user.email “你的邮箱”

  • 再输入npm install hexo-deployer-git –save，安装hexo git插件。
    单词一定不要打错了！！！

若输入后，返回的没有显示ERRO，只有WARN目前是没关系的。

  • 完成后，在$ 后输入hexo clean && hexo g && hexo d

如果这是第一次，终端会要求输入Github 的邮箱和密码，正确输入后，等待片刻，就会把博客上传至Github 了。以后在每次把博客写完后，执行一下这个命令就可以直接发布了
